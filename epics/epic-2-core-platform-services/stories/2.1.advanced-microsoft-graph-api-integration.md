- 429 Too Many Requests
- 502 Bad Gateway
- 503 Service Unavailable
- 504 Gateway Timeout

- 401 Unauthorized
- 403 Forbidden (with token refresh attempt)

**Client Errors (Log and alert):**
- 400 Bad Request
- 404 Not Found
- 422 Unprocessable Entity

**Permanent Errors (Fail fast):**

- 405 Method Not Allowed
